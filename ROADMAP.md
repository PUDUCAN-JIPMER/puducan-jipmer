# 🗺️ PuduCan Roadmap

> This roadmap reflects where the project is headed. It is not a strict commitment — priorities may shift based on contributor interest and real-world feedback from JIPMER.

---

## ✅ v1.1.0 — Stable (Released)

- Role-based dashboards for Admin, Doctor, Nurse, and ASHA workers
- Comprehensive patient management (add, view, edit, delete, recover)
- Advanced data table with search, filter, and pagination
- CSV / Excel import and export
- Real-time Firestore sync
- Light and dark theme support
- Mobile-friendly ASHA worker view

---

## 🔄 v1.2.0 — In Progress

- Improved test coverage with Vitest + React Testing Library
- Documentation overhaul (README, CONTRIBUTING)
- Better Firestore security rules
- General UI/UX polish and bug fixes

---

## 🔮 v1.3.0 — Planned

- [ ] Confirmation popup before recovering deleted patients (#37)
